# SuperStore US 2015 - Data Analysis Project

## üìä Project Overview
A comprehensive analysis of SuperStore's 2015 sales data in the United States, examining sales performance, profitability, regional trends, and customer behavior.

## üìÅ Dataset Description
The dataset contains transactional data for SuperStore US in 2015, including:

### Files/Sheets:
- **Raw Transaction Data** (`Table1_1`): 100+ detailed transaction records
- **Regional Sales Summary**: Sales by region (Central, East, South, West)
- **Product Category Profit**: Profit analysis by product category
- **Monthly Order Trends**: Order patterns by month
- **Top Customer Sales**: Sales performance by customer
- **Sales Trend Over Time**: Monthly sales patterns

### Key Columns:
- Order details (ID, priority, dates)
- Customer information (ID, name, segment)
- Product details (category, sub-category, name)
- Financial metrics (sales, profit, discount, unit price)
- Geographical data (region, state, city)
- Shipping information (mode, cost, time)
